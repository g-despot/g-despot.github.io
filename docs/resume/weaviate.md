# Weaviate

### Technical Writer and Developer Experience: January 2025 - Present

As a member of the **Developer Experience and Education team**, I enhance the learning journey for developers using the Weaviate vector database. My work focuses on:

- **Documenting new database, cloud, and agentic features** and actively testing them from a user perspective to provide feedback to engineering teams.
- **Developing and maintaining a Docusaurus (React) documentation platform** and optimizing it for use with LLMs, agents, and other AI tooling.
- **Creating practical code examples, tutorials, blog posts, and demo applications** to showcase Weaviate functionalities and best practices.
- **Developing an automated test framework** for validating documentation code integrity.
- **Providing community support and analyzing engagement metrics** to optimize the developer journey through the documentation.
- **Identifying and resolving developer friction points** by collaborating with Product, Engineering, and Community teams, as well as holding user interviews to validate the API design of client libraries and database features.

Programming languages/technologies used: Python, C++, Java, Erlang, C, Bash  
Tools used: Git, Gerrit, Docker, Wireshark, Eclipse, VSC
